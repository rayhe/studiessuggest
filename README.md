# Studies Suggest

Evidence-based publication featuring counterintuitive scientific findings that challenge conventional wisdom. Every study is verified, sourced with DOIs, and fact-checked against replication data.

🌐 **Live:** [studiessuggest.org](https://studiessuggest.org)

## What This Is

A collection of articles about scientific findings that contradict common beliefs. Each article:

- Features a real, published study with DOI and full citation
- Includes a Study Card with methodology, sample size, effect size, and replication status
- Rates counterintuitiveness on a 1-5 scale
- Provides actionable takeaways
- Includes audio narration (ElevenLabs TTS)
- Is evaluated against a 7-critic pipeline before publication

## Design

Nature-themed: earthy tones, organic shapes, serif headlines (Lora), clean sans-serif body (Source Sans 3). Credible research publication meets nature magazine.
